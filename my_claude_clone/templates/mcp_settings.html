💡 Как подключить Composio MCP?

  1. Открой app.composio.dev → войди / зарегистрируйся
  2. Перейди в Settings → API Keys → скопируй свой ключ
  3. URL сервера: https://mcp.composio.dev/composio/mcp/YOUR_API_KEY/sse
  4. Поле имя: Composio MCP, URL — твой персональный URL, Auth Token — оставь пустым
Важно: URL https://connect.composio.dev/mcp — есть общий адрес. Тебе нужен персональный URL с твоим ключом — иначе 403 Ошибка.
